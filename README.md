# MiniOS

MiniOS is a lightweight, directory-aware development environment for Web4 / LMLM projects.

## Features
- Directory-based environment management using `direnv`
- Nix flake integration for reproducible shells
- Shell hooks for bash, zsh, fish
- Optional NixOS modules for system-wide setup
